PHP en bref

  • PHP est un langager serveur cross-platform
  • Langage interprété qui s'éxecute sur un serveur
  • Permet de générer des pages HTML dynamique
  • La combinaison MYSQL et PHP permet d'extraire et de stocker des donnée dans une BDD

Historique

1994 : créé par Rasmus Lerdorf pour ses besoins personnels

Historique

  • 1995 : première version publique - gestion d'un compteur d'accés
  • 1995 : gestion de formulaire - livre d'or
  • 1997 : plus de 50000 sites Web utilisent PHP
  • 1997 : PHP3 projet équipe, réécriture complète de l'interpréteur

Historique

  • 1999 : 150 000 sites Web utilisent PHP3
  • 2000 : PHP4 le projet est maintenant chapeauté par Zend
  • Aujourd'hui, plus de 5 000 000 de sites Web dans le monde entier
  • 1997 : projet équipe, réécriture complète de l'interpréteur

Intéret du PHP

  • Très populaire et très utilisé
  • Très portable (fonctionne sous Windows/Unix...)
  • Syntaxe héritée du C, du shell et du Perl
  • Nombreux FAQ, tutoriaux, forum disponibles en ligne
  • Conçu pour fonctionner efficacement avec le serveur Apache lui aussi en open source

Commet ca marche

  • L'interpréteur de PHP est intégrée au serveur HTTP
  • Le serveur lit les instructions intégrées à la page HTML, interpréte ces instructions et remplace le résultat
  • Le client n'a pas accés au code au source et recoit uniquement le résultat

PHP en bref

  • un bloc PHP est un groupe de lignes encadré par deux balises
  • toute ligne située à l'extérieur de ces balises n'est pas interprétée : elle est recopiée à l'identique
  • toute ligne située à l'intérieur de ces balises est interprétée par le moteur comme une instruction PHP

Les bases du langage PHP

Types, chaînes de caractères, tableaux, opérateurs, instructions conditionnelles, boucles, fonctions, variables, objets
